# Icons

このフォルダにトップナビゲーション用のアイコン画像を配置してください。

## 必要なファイル

| ファイル名 | 用途 | 内容 |
|---|---|---|
| `nav_analytics.png` | 分析 | 電球＋虫眼鏡 |
| `nav_campaign.png` | 広告管理 | フォルダ |
| `nav_budget.png` | 予算設定 | ¥マーク（円） |
| `nav_settings.png` | 全体設定 | 歯車 |

## 推奨仕様

- 形式：PNG（透過背景）
- サイズ：128px × 128px 以上
- 4ファイル同じサイズ・比率推奨
